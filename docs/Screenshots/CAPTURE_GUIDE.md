# Screenshot Capture Guide — BrickBlast: Velocity Market

Capture all screenshots at the game's native 1200×867 logical resolution (or full-screen capture cropped to the game window).  
Save as PNG. Filename convention: `SS-##_description.png`.

## Required Screenshots (5 minimum)

| # | Filename | Screen to capture | What to show |
|---|----------|-------------------|--------------|
| 1 | `SS-01_main_menu.png` | Main Menu | Title, Play/Store/Settings/Credits buttons, version number |
| 2 | `SS-02_gameplay_level3.png` | Gameplay — Level 3+ | Ball in motion, multiple brick types, HUD with score + coins + lives |
| 3 | `SS-03_store_balls.png` | Store — Balls tab | Owned / equipped / purchasable ball skins, coin balance |
| 4 | `SS-04_store_bonuses.png` | Store — Bonuses tab | Bonus-pack selection grid, themed icons |
| 5 | `SS-05_game_over.png` | Game Over overlay | Score, coins earned, Retry / Store / Menu buttons |
| 6 *(bonus)* | `SS-06_level_complete.png` | Level Complete screen | Score tally, coins earned, Next Level button |
| 7 *(bonus)* | `SS-07_credits.png` | Credits screen | Team names, technology, starter source |
| 8 *(bonus)* | `SS-08_settings_sync.png` | Settings (Options) | Sync status label, manual sync button, volume sliders |

## How to capture

1. Run `BrickBlast.exe` from `bin\Release\net10.0-windows10.0.22000.0\`.
2. Navigate to each screen.
3. Press **Win + Shift + S** (Snipping Tool) or **PrintScreen** and crop to the game window.
4. Save to this folder: `Docs/Screenshots/`.
5. Update `Docs/Submission/FinalSubmissionChecklist.md` items 4.13, 4.14, 5.3 with evidence paths.
