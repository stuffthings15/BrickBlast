# Trailer Capture Guide — BrickBlast: Velocity Market

## Deliverable

| Property | Value |
|----------|-------|
| Filename | `BrickBlast_Trailer_v1.mp4` |
| Format | MP4 (H.264) |
| Resolution | 1920 × 1080 landscape (or 1200 × 867 game window + side panels) |
| Length | 30 – 60 seconds |
| Audio | In-game music + SFX (or post-produced audio) |

Place the finished file in this folder: `Docs/Trailer/`.

---

## Recommended Recording Tool

**Windows Game Bar** (built-in):
1. Focus the BrickBlast game window.
2. Press **Win + G** → Record → Start.
3. Play through the scenes listed below.
4. Stop recording — clip saved to `Videos\Captures\`.
5. Trim and export to MP4 with any free editor (DaVinci Resolve, Kdenlive, CapCut).

**OBS Studio** (free, https://obsproject.com):
1. Add a Window Capture source targeting `BrickBlast`.
2. Set canvas to 1920×1080 (letterbox if needed).
3. Start recording, play scenes, stop, and export.

## Shot List (30–60 s)

| Time | Scene | Notes |
|------|-------|-------|
| 0–3 s | Title card | "BrickBlast: Velocity Market" on dark starfield — let the game sit on the main menu |
| 3–10 s | Gameplay clips | Ball launch, paddle movement, brick destruction combos |
| 10–18 s | Brick types | Show Standard → Durable (taking multiple hits) → Reward → Hazard → Moving bricks |
| 18–25 s | Marketplace | Open Store, scroll Balls/Bricks/Bonuses tabs, purchase an item, equip it |
| 25–35 s | Level progression | Jump from Level 1 to Level 5+ showing increasing density |
| 35–42 s | Sync / networking | Show HUD sync label (Online → Synced), Settings sync button |
| 42–52 s | Polish montage | Screen shake, particle bursts, power-up collection, Game Over screen |
| 52–60 s | Call to action | Return to main menu — freeze on "Play BrickBlast: Velocity Market  v1.0.0" |
