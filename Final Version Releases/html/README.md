# BrickBlast — HTML Release

This folder is the standalone browser release of BrickBlast.

## Running Locally

Open `index.html` in any modern browser, or serve with a local web server:

```bash
# Python 3
python -m http.server 8080
# then open http://localhost:8080

# Node (npx serve)
npx serve .
```

## Deployment

Upload the entire folder contents to any static host:
- **itch.io** — zip the folder, upload as HTML game, set viewport 1200×867
- **GitHub Pages** — push to a `gh-pages` branch
- **Netlify / Vercel** — drag-and-drop the folder

## Controls
`← →` or `A D` — move paddle · `SPACE` — start/resume · `F` — speed boost · `H/O` — options
