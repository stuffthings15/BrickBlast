# BrickBlast — React Native Release

Cross-platform React Native project wrapping the HTML5 game via `WebView`.

## Structure
| Path | Purpose |
|------|---------|
| `App.tsx` | Root component with `WebView` game wrapper |
| `package.json` | RN project manifest |
| `android/` | Android native project (generated by `npx react-native init`) |
| `ios/` | iOS native project (generated by `npx react-native init`) |
| `assets/` | Game assets for offline embedding |

## Setup

```bash
npm install

# Android
npx react-native run-android

# iOS (macOS only)
cd ios && pod install && cd ..
npx react-native run-ios
```

## Release Builds

**Android APK/AAB:**
```bash
cd android
./gradlew assembleRelease      # APK
./gradlew bundleRelease        # AAB (Google Play)
```

**iOS:**
Open `ios/BrickBlast.xcworkspace` in Xcode → Product → Archive → Distribute

## Requirements
- Node.js 18+
- React Native CLI (`npm install -g react-native-cli`)
- Android Studio (for Android) or Xcode 15+ (for iOS)
- `react-native-webview` package (already in `package.json`)
