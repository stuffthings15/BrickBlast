# Brick Blast — HTML Browser Version

## How to Run
**Double-click `RUN_HTML.bat`** to open the game in your default browser.

### Alternative
Open `web/index.html` directly in any modern browser.

### Hosted Version
If deployed to GitHub Pages, visit: `https://stuffthings15.github.io/BrickBlast/web/`

## Requirements
- Any modern browser (Chrome, Firefox, Edge, Safari)
- No installation needed

## Controls
| Input | Action |
|-------|--------|
| ← → / A D / WASD | Move paddle |
| SPACE / Click / Tap | Start / Resume |
| P / ESC | Pause |
| F | Speed boost (2x) |
| H / O | Options menu |
| Touch drag | Move paddle (mobile) |
| Gamepad | Full controller support |

## Platform
- **Type:** Single-file HTML5/Canvas/JavaScript
- **Source:** `web/index.html`
- **Works on:** Windows, Mac, Linux, Chromebook — any device with a browser
