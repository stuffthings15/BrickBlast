# Brick Blast — HTML Browser Version

## How to Run
**Double-click `RUN_HTML.bat`** or open `index.html` directly in any browser.

This folder is fully self-contained. Zip it and share — works on any device with a browser.

## Contents
| File / Folder | Purpose |
|---------------|---------|
| `index.html` | The complete game (single file, HTML5 / Canvas / JavaScript) |
| `manifest.json` | PWA manifest for "Add to Home Screen" install |
| `icons/` | App icons for PWA install |
| `assets/` | Game sprites, tiles, and UI images |
| `RUN_HTML.bat` | Shortcut to open in browser (Windows) |
| `README.md` | This file |

## Requirements
- Any modern browser (Chrome, Firefox, Edge, Safari)
- No installation needed

## Controls
| Input | Action |
|-------|--------|
| ← → / A D / WASD | Move paddle |
| SPACE / Click / Tap | Start / Resume / Speed boost (2x) |
| P / ESC | Pause |
| F | Speed boost (2x) |
| H / O | Options menu |
| Touch drag | Move paddle (mobile) |
| Gamepad | Full controller support |

## Platform
- **Type:** Single-file HTML5/Canvas/JavaScript
- **Source:** `web/index.html`
- **Works on:** Windows, Mac, Linux, Chromebook — any device with a browser
